## Description
Just a test project using NestJS, GraphQL and TypeORM

## Installation with docker compose

```bash
$ docker compose up
```

## Installation without docker

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
